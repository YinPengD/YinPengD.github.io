MySQL数据库的基础操作

Posted on 2017-09-28 | In 数据库

MySQL是一个数据库，便于程序操作，储存百万以上级别的数据，是关系型数据库:是建立在关系模型基础上的数据库，借助集合代数等数学概念和方法来处理数据库中的数据

主键（Primary Key)
  • 每行数据独一无二的标识
  • 一个表必须有主键（只能有一个主键）
  • 不循环为NULL（空值）
  • 由一列或者多列组成
Unique Key
  • 数据不能重复
  • 允许一条为NULL
外键（Foreign Key)
  • 这列数据引用了另一个标的主键

当数据之间有多对多关系的时候可以创建一个中间表来链接两者关系

例如玩家的角色与物品之间，角色可以拥有许多个，物品也可以拥有很多

插入表：

insert into tablename(col_name,colname,col_name)
values
(value1,value2,value3,value4,value5);
查询数据 : select * from mytable;
查询前4条数据：select * from mytable limit 4;
查询从2条开始的3条数据： select * from mytable limit 2,3;
查询某一列数据：select username from mytable;
查询某一列大于某个数的数据： select username from mytable where id>8;
对查询进行排序： select * from mytable order by id;
对查询进行逆排序： select *from mytable order by desc;
排序后进行限值：select * from mytable order by id desc limid 1;
两列数据进行排序：select * from actor order by first_name,last_name;
在查询表头后加as加要修改的名字可以修改查询的表头：select last_name as LastName,first_name FirstName for actor;

查询静态值：

select ‘some string’;
select 1+1’
select now();

select curdate();
select curtime();
select pi();
select mod(45,7); //求余
select sqrt(25);

查询可以使用的功能函数：

round(): 四舍五入；
round(columnname,x):四舍五入保留x位小数
floor() 直接舍去小数部分
ceiling() 直接入 小数部分

运算：

/ （除 结果为浮点数）
div (除 结果为整数）
% mod (求余，结果为浮点数）
+ - 加减

字符串操作：

链接数据：concat
取数据的前几位：left
取数据的长度：length
倒转数据：reverse
替换数据：replace

格式化输出数据：date_format %m %d %b %y %T
ditinct： 去除重复的数据 select distinct name from mytable;

判断条件：

数字 > < = >= <= <>
字符串 = ‘ ’ > < = >= <= <> !=
逻辑操作：
is仅用于判断是否为空
and or not (and 优先级>or)
实例：

范围判断：

in (not in)： 在某个范围之间
between (not between): 在两个范围之间；
like (not like):查询符合某个添加的(模糊搜索）

实例：

更新数据

update tablename set col_name = value,col_name = value where codition;

删除数据

delete from tablename where condition;
